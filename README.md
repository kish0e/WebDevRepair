# Web Development Repair assessment
This is the boilerplate code to give you headstart on the repair assessment.
Use the project description provided as PDF as your requirement checklist to see what needs to be implemented. 

## Setup

The project allready has: 
- .NET 
- EF Core using SQLite
- A database context + models
- A Frontend using React + Parcel for bundling and compiling your components

Run the following commands to get started. 
- `dotnet ef database update`
- `dotnet watch run`
- `npm run watch`

Feel free to make any changes to the setup and configuration to your personal preferences.
